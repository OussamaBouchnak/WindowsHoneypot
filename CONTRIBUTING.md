# Contributing

Thanks for your interest in contributing to this project!

## How to Contribute
- Fork the repository
- Create a new branch from `main`
- Make your changes with clear, concise commits
- Open a Pull Request describing what you changed and why

## Guidelines
- Keep changes small and focused
- Follow existing code style and structure
- Add comments or documentation when needed

## Issues
- Feel free to open an issue for bugs, improvements, or suggestions
- Please provide clear steps to reproduce bugs

## Code of Conduct
By contributing, you agree to be respectful and constructive in discussions.
